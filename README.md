# Data Reporting Automation

A Python portfolio project focused on automated reporting workflows using synthetic sample datasets.

This repository demonstrates practical data handling, report generation and workflow automation relevant to finance operations, reporting analyst and data analyst roles.

## Project Purpose

The aim of this project is to automate structured reporting processes using Python and pandas.

The project demonstrates:

- CSV data ingestion
- Automated KPI calculation
- Summary report generation
- Data cleaning and formatting
- Exporting structured outputs
- Reporting workflow automation

## Tools Used

- Python
- pandas
- NumPy

## Repository Structure

```text
data-reporting-automation/
├── README.md

├── reporting_automation.py

├── .gitignore
└── outputs/
    └── README.md
```

## How To Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the automation script:

```bash
python reporting_automation.py
```

Generated outputs are saved into the `outputs/` folder.
